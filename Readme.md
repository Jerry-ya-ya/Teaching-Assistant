# 前言

大家好，我是這學期的助教。

在這門課裡，我和杰克會帶著大家從零開始，逐步建立屬於自己的網站專案。

我們的目標不只是完成課堂作業，而是實際打造一個可以部署到公網、真正對外公開的網站作品。
這個網站不會只是期末交完就結束的作業，而是一個可以持續擴充、無限 scale up 的專案。

如果時間允許，我們甚至可以把它進一步做成 Web App，讓你的網站不只是被瀏覽，而是可以像 App 一樣放在手機桌面上，成為真正可使用的產品。

在這個 AI 快速發展的時代，我認為比起單純記住語法，更重要的是——
能夠建立好自己的開發環境、看得懂自己真正需要的程式碼，還有培養清楚的設計思維。

現在 AI agent 很盛行，像是 Cursor、各種所謂的「vibe coding」工具都非常方便。
但我還是希望大家，在使用這些工具之前，能夠先自己動手試著理解問題、拆解問題、讀懂錯誤訊息，並嘗試剪接出真正適合自己專案的程式碼。

如果你真的花了一段時間，例如想了一個小時還是卡住，那再善用 AI agent 來幫助你，我覺得這會是一種更健康的學習方式。

工具可以幫助我們加速，但如果沒有理解基礎，我們很容易被工具帶著走。
我希望這學期大家不是「寫出程式」，而是真的「看得懂自己寫了什麼」。

我自己其實也還在學習的路上，不過或許可以帶大家避開一些我以前踩過的坑。
也很希望大家多和我討論問題，我相信我也能從你們身上學到不少新的觀點。

如果你們在學習過程中遇到困難，請不要猶豫來找我。我很期待和大家一起解決問題，一起成長。

也謝謝杰克給我這個機會，讓我能夠在這門課中持續學習與精進。

希望這學期我們都能做出讓自己驕傲且可以做為面試資本的作品。
